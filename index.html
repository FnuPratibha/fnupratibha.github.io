Publications & Research

Integrating Reconfigurable Accelerators with Quantum Computing

FNU Pratibha, Parth Ganeriwala, Naveed Mahmud

IEEE International Conference on Quantum Computing and Engineering (QCE25), Q-CORE 2025

Proposes a framework that integrates FPGAs with quantum processing units (QPUs) for hybrid quantum–classical (HQC) algorithms. Classical subroutines are accelerated on FPGA using a high-throughput pipeline, while quantum subroutines execute on QPUs. For variational quantum classification on MNIST, the FPGA pipeline achieves up to 8× speedup over a state-of-the-art quantum software framework running on a server-grade CPU.

A Reconfigurable Framework for Hybrid Quantum–Classical Computing

FNU Pratibha, Naveed Mahmud

Algorithms (MDPI), 2025

Presents a reconfigurable framework that couples FPGAs and QPUs to accelerate hybrid quantum–classical workloads by offloading classical subroutines to a high-throughput FPGA pipeline. Demonstrates up to 8× runtime improvement over a CPU baseline for a variational quantum classification algorithm.

High-Level Acceleration of Quantum Simulation Frameworks on Reconfigurable Hardware

FNU Pratibha, Vinayak Jha, Ishraq Islam, Anshul Maurya, Manu Chaudhary, Alvir Nobel, Naveed Mahmud, Esam El-Araby

Design Automation Conference (DAC), 2025

Hybrid Quantum–Classical Machine Learning for Sentiment Analysis

International Conference on Machine Learning and Applications (ICMLA), 2023
